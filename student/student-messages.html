Dear John,

I hope this message finds you well. I wanted to provide you with the updated guidelines for the database design project.

Project Requirements:
  • Design a database for a university library system
  • Include at least 5 entities with appropriate relationships
  • Implement the database using MySQL
  • Create at least 10 sample queries demonstrating CRUD operations
  • Submit ER diagram, SQL scripts, and query results
Important Dates:
  • Project proposal due: March 20, 2024
  • ER diagram submission: April 5, 2024
  • Final project submission: May 1, 2024

Please let me know if you have any questions about the project requirements. I'm available during my office hours (Mon/Wed 2-4 PM) or by appointment.
